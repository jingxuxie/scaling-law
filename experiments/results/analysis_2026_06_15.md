# Analysis of filter and alignment sweeps

This note summarizes the first local sweeps in `experiments/results`.

## Runs checked

The filter sweep used

```text
dimension = 300000
n_range = [10, 10000]
num_n = 50
a_values = 1.25, 1.5, 2.0
b_values = 1.2, 1.4, 1.8
rho_values = 1e-16, 1e-12, 1e-8
wd_values = 0, 1e-4, 1e-3, 1e-2
```

and produced 108 rows.  The alignment sweep used

```text
alignment_dimensions = 256, 512, 1024
a_values = 1.25, 1.5, 2.0
rho_values = 1e-12, 1e-8
include_random = true
```

and produced 18 rows.

## Main result: learned-count exponents pass

The learned-count exponents match very well.

Examples from the filter sweep:

```text
a=1.25: SGD count 0.807 vs 0.800, Adam count 1.586 vs 1.600
a=1.50: SGD count 0.681 vs 0.667, Adam count 1.334 vs 1.333
a=2.00: SGD count 0.520 vs 0.500, Adam count 1.004 vs 1.000
```

This supports the central exponent prediction:

\[
    K_{\rm SGD}(n)\asymp n^{1/a},
    \qquad
    K_{\rm Adam/RMSProp}(n)\asymp n^{2/a}
\]

in the active pre-damping regime.

AdamW count slopes also match the horizon-ceiling prediction.  For instance, with \(a=1.25\):

```text
wd=0:     AdamW count 1.586 vs 1.600
wd=1e-3: AdamW count 1.346 vs 1.346
wd=1e-2: AdamW count 0.308 vs 0.308
```

This supports

\[
    K_{\rm AdamW}(n)\asymp K_{\rho,1/2}(\min\{n,\lambda_{\rm wd}^{-1}\}).
\]

## Bias slopes: good in the valid regime, diagnostic outside it

The SGD bias slopes are generally close to the clean prediction.  Adam/RMSProp bias slopes are close when \(b\) is in the clean regime

\[
    1<b<a/2+1.
\]

For example:

```text
a=1.5, b=1.4: Adam bias -0.566 vs -0.533
a=2.0, b=1.4: Adam bias -0.414 vs -0.400
a=2.0, b=1.8: Adam bias -0.725 vs -0.800
```

The larger deviations occur when \(b\) is outside or near the clean regime.  For example:

```text
a=1.25, b=1.8: clean prediction -1.280, observed about -0.924
```

Here \(a/2+1=1.625\), so \(b=1.8\) is in the smooth-source regime.  The clean formula \(n^{-2(b-1)/a}\) should not be used.  The correct smooth-source theorem gives saturation at slope \(-1\), up to finite-size and logarithmic effects.

This motivated `notes/12_smooth_source_bias_saturation.md`.

## Alignment results pass strongly

The coordinate-alignment sweep is very clean.  Examples:

```text
a=1.25: aligned slope -0.625, flat slope -1.250, random slope about -1.263
a=1.50: aligned slope -0.750, flat slope -1.500, random slope about -1.526
a=2.00: aligned slope -1.000, flat slope -2.000, random slope about -2.05
```

This supports the coordinate-alignment theorem:

\[
\begin{array}{lll}
\text{aligned coordinates} & \Rightarrow & q_{\rm eff}=1/2,\\
\text{flat/global mixing} & \Rightarrow & q_{\rm eff}=0 \text{ at exponent level.}
\end{array}
\]

The random orthogonal cases are closer to the flat/SGD-like spectrum than to the aligned spectrum.  This is exactly the concern that matters for Gaussian-sketched and random-feature models.

## Conclusion

The sweeps support the central story:

1. Count exponents match the theory very well.
2. AdamW horizon saturation is visible and matches the theory.
3. Coordinate alignment is decisive: aligned coordinates show \(a\mapsto a/2\), while flat/random-mixed coordinates do not.
4. Bias-slope discrepancies are informative rather than fatal: they identify the smooth-source saturation regime missing from the earlier clean-regime theorem.
